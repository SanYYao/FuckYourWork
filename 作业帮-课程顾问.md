## 个人评估

### 匹配度

- 技能匹配：7/10（可学，但销售是弱项）
- 发展空间：8/10（校长是目标，晋升通道清）
- 薪资水平：8/10（8k-12k，郑州市场不错）
- **综合评分：7.5/10**

### 优点 ✅

1. **大公司背书** → 简历加分，品牌认可
2. **福利超全** → 七险一金 + 免费滴滴 + 体检 + 团建
3. **晋升通道清** → 组长-主管-经理-校长，目标明确
4. **客户资源现成** → 不需自己开发，压力降低
5. **薪资天花板高** → 8k-12k 上不封顶，绩效驱动
6. **新人保护** → 前 2 个月 1k 保底，风险低
7. **工作时间合理** → 13-22 常规，有周末休息

### 缺点 ⚠️

1. **销售驱动岗** → 核心工作是"转化"，INFP 可能不适
2. **需要外向** → 岗位明确要求"性格外向"，社恐困境
3. **出镜要求** → 需要讲题视频出镜，隐私曝光
4. **心理阴影** → 上次销售岗失败的创伤
5. **教育行业竞争** → 学科功底竞争激烈，需持续学习
