# FuckYourWork 🚀

> **数据驱动的工作搜索平台** — 为郑州求职者提供冷静、理性、不讨好的岗位分析

**不是炫耀简历的地方，而是解剖工作陷阱的地方。**

---

## 项目简介

这是一个使用 **VitePress** 构建的工作搜索和分析平台，聚焦郑州地区。与其他招聘网站不同，FuckYourWork 提供的不是包装好的岗位描述，而是：

- ✅ **冷硬的优缺点分析** — 不隐瞒坑
- ✅ **市场行情对标** — 薪资是否被压低
- ✅ **个人匹配度评分** — 诚实的风险提示
- ✅ **文化压力评估** — 团队气氛和加班文化
- ✅ **晋升路线调查** — 这个岗位能带你去哪里
- ✅ **面试提问清单** — 问关键问题，别被话术骗

---

## 项目特点

### 1. **数据结构清晰**
```
每个岗位包含：
├── 基本信息（公司、地点、薪资、来源）
├── 薪酬分析（底薪、补贴、绩效、平均实际收入）
├── 福利待遇（七险一金还是三险一金）
├── 任职要求分析（匹配度 & 风险评估）
├── 关键风险评估（性格、销售压力、工作强度）
├── 个人评估（综合评分 0-10）
├── 面试提问清单（关键问题）
└── 个人备注（真话评价）
```

### 2. **自动化工作流**
```bash
# 三步推送到线上
cp /FuckYourWork/{company}-{position}.md /FuckYourWork-git/docs/
cd FuckYourWork-git
git add . && git commit -m 'message' && git push
# Cloudflare Pages 自动部署 ✓
```

### 3. **Cloudflare Pages 自动部署**
- 推送到 GitHub → Cloudflare 自动构建 + 部署
- 无需手动操作，即时生效
- 支持自定义域名

## 使用指南

### 1. **本地开发**
```bash
npm install
npm run docs:dev  # 启动开发服务器 localhost:5173
```

### 2. **添加新岗位**
1. 在 `/docs/` 目录创建 `{公司}-{岗位}.md`
2. 使用提供的模板结构
3. 填写完整的分析信息
4. 更新 `docs/index.md` 的链接
5. 执行 git 推送命令自动部署

## 哲学

### 为什么要做这个项目？

郑州招聘市场充满了：
- 🚩 虚假的福利承诺（"包住"实际是宿舍）
- 🚩 隐瞒的高压文化（"快速成长"=加班996）
- 🚩 精心包装的坑（美工岗要求C4D但实际就是PS）
- 🚩 被动接受的现状（求职者缺乏议价权）

**FuckYourWork** 的目标很简单：**还原真相，找到真正值得的机会。**

不是为了讽刺工作本身，而是为了讽刺那些试图欺骗求职者的虚伪包装。

### 核心原则

1. **数据优于感觉** — 用市场行情对标，别被HR忽悠
2. **风险提前说** — 优点缺点一样重要
3. **个人评分制** — 不同人的适配度不同，评分基于真实背景
4. **面试提问清单** — 不要被话术带走，问关键问题
5. **长期评估** — 入职3-6个月后回顾，更新真实反馈
